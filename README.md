# Honey Crumb

**Тема:** Минималистичный адаптивный сайт для кафе

**Описание проекта:**  
Honey Crumb — это минималистичный сайт для кафе, разработанный с акцентом на удобство пользователей. Сайт показывает меню, информацию о кафе, контакты и позволяет управлять заказами через страницу "Корзина". Он полностью адаптивен: корректно отображается на мобильных устройствах, планшетах и десктопах.  

**Страницы сайта:**  
1. **Главная** — приветствие, краткая информация о кафе и акции.  
2. **Меню** — список блюд и напитков с описанием и ценами.  
3. **О нас** — информация о кафе, философия и история заведения.  
4. **Контакты** — адрес, телефон, карта и форма обратной связи.  
5. **Корзина** — страница для просмотра и управления выбранными заказами.  

**Используемые технологии:**  
- **HTML5** — разметка сайта, семантические теги для удобства навигации и SEO.  
- **CSS3** — стилизация сайта, адаптивность с использованием media queries, эффекты для кнопок и карточек меню.  
- **JavaScript** — небольшие скрипты для интерактивности, например, открытие модальных окон и простая валидация форм.  

**Особенности проекта:**  
- Адаптивный дизайн mobile-first.  
- Минималистичный интерфейс для удобства пользователей.  
- Легкость расширения и поддержки новых страниц или функций.
